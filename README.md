# calc_and_time
MCP calc_and_time

为模型提供时间和简单计算工具的MCP工具

MCP tools that provide time and simple calculation utilities for the model

## tools

calculate_expression
计算数学表达式，支持加减乘除、括号、百分比等常用计算
unit_conversion
单位换算，支持长度、温度、面积等单位之间的转换
get_current_datetime
获取当前日期与时间
timestamp_conversion
时间戳与日期时间相互转换
date_arithmetic
日期加减运算，支持对日期进行年、月、日、时、分、秒的加减操作
date_difference
计算两个日期之间的差值，支持按天、小时、分钟、秒为单位返回
generate_random_numbers
生成随机数，支持定义最大最小、生成数量、保留几位小数
